# Vanguard Ops Platform — Current State

## What This Is
Operations platform demo for Vanguard Renewables. Organic waste dispatch, BOL/COD management, facility tracking, compliance, billing. Built to close a deal with a BlackRock-backed company scaling to 150+ facilities.

## Production URL
https://vanguard-ops-mu.vercel.app

## GitHub
https://github.com/RodolfoDavidAlvarez/vanguard-ops

## Tech Stack
Next.js 16.2.2 (App Router), Tailwind v4, Recharts, Lucide React, Vercel

## Current State
- Demo with realistic mock data (8 facilities, 42 loads, 25 CODs, 10 customers)
- 10 pages: Dashboard, Dispatch, Loads, CODs, Facilities, Customers, Billing, Compliance, Notifications, Settings
- No backend, no auth, no real data — purely a polished frontend demo
- Deployed on Vercel (BSA account)
